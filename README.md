# IPL-Data-Analysis
IPL Data Analysis using Python, SQL, Pandas and Data Visualization

## Tools Used
- Python
- Pandas
- SQL
- Matplotlib
- Google Colab

## Analysis Performed
- Team with most wins
- Top run scorers
- Top wicket takers
- Toss impact on match results

## Dataset
IPL dataset from Kaggle:https://www.kaggle.com/datasets/patrickb1912/ipl-complete-dataset-20082020/data
